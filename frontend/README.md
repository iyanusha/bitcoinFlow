# BitcoinFlow Frontend

React + Vite frontend for the BitcoinFlow sBTC stacking vault.

## Setup

```bash
npm install
npm run dev
```

## Features

- Stacks wallet connection via @stacks/connect
- sBTC deposit and withdrawal through contract calls
- Vault statistics dashboard
- Flow token balance tracking

## Tech Stack

- React 19 with TypeScript
- Vite for build tooling
- @stacks/connect for wallet integration
- @stacks/transactions for contract calls
